##  Project Overview

Task Manager is a full-stack web application that helps organizations manage interns, projects, and assigned tasks efficiently.

The system supports role-based access:

- **Admin**
  - Create and manage projects
  - Assign tasks to interns
  - View intern submissions
  - Monitor active interns
  - Track task progress

- **Intern**
  - View assigned tasks
  - Update task status
  - Upload task submission files
  - Track personal task progress

The application includes authentication, file upload functionality, and role-based dashboards.
